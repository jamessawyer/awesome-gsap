---
title: scrollor Property
---


📚 定义：

`.scrollor: Element | window` 与ScrollTrigger相关联滚动元素（或 `window`）。
- 只读
- 与ScrollTrigger相关联的滚动条容器。默认是 `window` (即 viewport)
- 返回与ScrollTrigger相关联的滚动条容器 Element | window

📚 定义：

`.autoScroll: Number` 容器滚动的速度

- 为了开启Draggable被拖动到可滚动容器 `40px` 位置时，容器滚动的速度，
- 设置一个非0值，`1` 表示正常速度；`2` 表示2倍速
- 为了更直觉或自然的感觉，越靠近边缘，滚动速度越快
- 默认值为 `0`, 即没有自动滚动
